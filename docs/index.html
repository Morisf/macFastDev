Macfastdev

Short and simple script which I create to use on my job to make all mac at office with same configuration

Download this project as a .zip file Download this project as a tar.gz file

MacOS fast dev env installation

What is in

  • install x-code
  • install brew

  • Packages

    Package name Version
    GIT @latest
    PHP v7.0
    Apache v4.6.1
    NodeJs @latest from NPM
    Gulp @latest from NPM
    Bower v2.4
    DNSMasq @latest

How it works

  1. Clone files
  2. Open terminal
  3. run sh ./install.sh
  4. give root password
